Question-1: What are the different ways to select an element in the DOM?

There is many ways to select an element in the DOM. We can select an element by using GetElementByID, GetElementsByClassName, GetElementByTagName, QuerySelector, QuerySelectorAll.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element instead of adding listeners to multiple child elements. It works because of event bubbling, which allows events to propagate up the DOM tree.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the dom where an event triggered on a child element "bubbles up" through its parent elements, continuing up to the root of the document unless stopped.

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element by using document.createElement() in JS. We can add any element in the document by using append() or appendChild() in JS. Also, We can remove any element from a document by using remove() or removeChild() in JS.
